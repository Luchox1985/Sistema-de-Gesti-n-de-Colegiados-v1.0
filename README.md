# Sistema de Gestión de Colegiados

Automatización del proceso de clasificación y gestión de colegiados
del Colegio Profesional (7000+ registros activos).

## ¿Qué hace?

- Lee automáticamente desde archivo Excel con openpyxl
- Menú interactivo de navegación (while + input)
- Ver padrón completo de colegiados
- Filtrar morosos — los que requieren gestión de cobranza
- Filtrar activos — los que van en nómina del seguro
- Contador automático por categoría

## Historial de versiones

- v1.0 → Sistema base: clasificación y alertas por estado
- v1.1 → Menú interactivo + filtros + contadores por estado
- v1.2 → Lectura directa desde Excel con openpyxl ← actual

## Roadmap

- v2.0 → Cruce automático de nóminas con aseguradoras
- v3.0 → Generación de reportes PDF
- v4.0 → Envío automático de correos por estado
- v5.0 → Base de datos SQL + interfaz de usuario

## Tecnologías

- Python 3.x
- openpyxl

## Contexto

Proyecto desarrollado como parte de la ruta de formación en
Ingeniería en Informática — Santo Tomás Online.
Aplicado al trabajo real de gestión de colegiados profesionales
del Colegio de Dentistas de Chile.
